# API Endpoints

## HTML API

### Root

 - `GET /` - loads React web app

## JSON API

### Users

 - `POST /api/users`
 - `PATCH /api/users`
 - `GET /api/users/:id`
 - `GET /api/users`

### Session
 - `POST /api/session`
 - `DELETE /api/session`

### Workouts

 - `GET /api/workouts`
 - `POST /api/workouts`
 - `GET /api/workouts/:id`
 - `PATCH /api/workouts/:id`
 - `DELETE /api/workouts/:id`

### Stats
 - `GET /api/stats`

### Comments(bonus)
 - `GET /api/workouts/:id/comments`
 - `POST /api/workouts/:id/comments`
